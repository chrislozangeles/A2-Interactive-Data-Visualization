# Corporate Ownership and Housing Structure in Boston (2004–2024)

This project analyzes patterns of corporate housing ownership across Boston neighborhoods using visualization techniques.

The analysis explores:

• Temporal trends in corporate ownership  
• Spatial variation across neighborhoods  
• Relationships between ownership, vacancy, and demographic composition
